Perceptron

Introduction

Perceptron is kind of supervised learning for binary classifier. It is an algorithm for learning the threshold function.

Model building

Assume we have a linearly separable data set, ensuring that the algorithm can converge. Our objective is to separate the sample points using a hyperplane, whose normal vector is determined by minimizing the loss function associated with the misclassified points. It should be noted that this solution is not unique. \[ \begin{align*} \sum_{i=1}^{k+1}i & = \left(\sum_{i=1}^{k}i\right) +(k+1)\\ & = \frac{k(k+1)}{2}+k+1 & (\text{by inductive hypothesis})\\ & = \frac{k(k+1)+2(k+1)}{2}\\ & = \frac{(k+1)(k+2)}{2}\\ & = \frac{(k+1)((k+1)+1)}{2}. \end{align*} \]
