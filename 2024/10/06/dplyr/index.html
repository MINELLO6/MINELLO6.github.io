The pipe

The first argument of all dplyr functions is a data frame or tibble. Dplyr uses the operator from magrittr which turns x into f(x, y). The hot key for it is ctrl + shift + M.

filter(): select a subset of rows in a data frame.

arrange(): reorder the rows

slice(): index rows by integer locations

select(): used to zoom in on a useful subset of a large datasets

mutate(): add new columns that are functions of existing columns

relocate(): change the order for data frame
